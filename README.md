# 🎬 TollyFlix

A Netflix-style movie discovery web app for Telugu, Hindi, Tamil, Malayalam & Kannada movies and web series — built with vanilla HTML, CSS & JavaScript using the TMDB API.



---

## ✨ Features

- 🎬 Browse Movies & 📺 Web Series by language
- 🔍 Search movies and series
- 🎭 Filter by Genre, Year, Rating, Sort
- 🔥 Trending, ⭐ Top Rated, 🆕 New Releases, 🔔 Upcoming
- 🎬 Official Trailers (YouTube embed)
- 📺 OTT Platform info — Netflix, Prime, Hotstar, ZEE5, Aha & more
- 🎵 Spotify songs link
- 🎭 Cast & Crew with person details
- ❤️ Favorites & 🕐 Watch Later (localStorage)
- 📜 Watch History
- 🎲 Surprise Me (random suggestion)
- ⭐ User Star Ratings
- 🌙 Dark / ☀️ Light Mode
- 👤 Login & Signup (localStorage based)
- 📱 Mobile Responsive
- ⚡ PWA Support (installable)

## 🚀 Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/tollyflix.git
   ```
2. Open `index.html` in your browser — no build step needed!

## 📁 Project Structure

```

├── login.html       # Login & Signup page
├── profile.html     # User profile page
├── script.js        # Main app logic
├── style.css        # Styles
├── logo.png         # App logo
└── manifest.json    # PWA manifest
```
